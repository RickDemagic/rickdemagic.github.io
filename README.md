# RD Systems

Moderne, strukturierte Business-Websites für regionale Unternehmen.

## Konzept

RD Systems steht für Klarheit, technische Präzision und visuelle Stärke.  
Keine unnötigen Frameworks. Kein Marketing-Blabla.  
Sauberer Code. Klare Architektur. Nachhaltige Performance.

---

## Technischer Aufbau

Dieses Projekt basiert auf einer klassischen, performanten Struktur:

- index.html – Struktur und Content
- style.css – Layout, Design, Animationen
- script.js – Scroll-Animation & Parallax
- hero.jpg – visuelles Key-Element im Hero-Bereich

---

## Features

- Fullscreen Hero mit Parallax-Effekt
- Scroll-Animationen via IntersectionObserver
- Responsive Layout (Mobile-first)
- Saubere CSS-Struktur ohne Framework-Abhängigkeiten
- Kontaktformular via Formspree (POST-basiert)

---

## Philosophie

Struktur vor Spielerei.  
Technik vor Marketing-Floskeln.  
Funktion vor Effekthascherei.
